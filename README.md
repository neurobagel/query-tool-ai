## Getting started

- clone the repo

  ```git clone https://github.com/neurobagel/query-tool-ai.git```
- create virtual environment
   
   `python3 -m venv venv`
   
   `source venv/bin/activate`

- complete installations 
 
   `pip install -r requirements.txt`

- pull the required ollama model
   
   ```ollama pull mistral```
